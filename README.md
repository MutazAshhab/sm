## NOTE: execute all commands from the root of the project and only change directory if instructed

## Install dependencies

Run the following:

To install backend deps

```bash
pnpm install
```

To install frontend deps

```bash
cd ./frontend
pnpm install
```

## Running the application dependencies

Run the following:

To run the backend

```bash
pnpm start-backend
```

To run the frontend

```bash
cd ./frontend
pnpm dev
```
